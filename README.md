## What is this?

A clean, no-nonsense music player that lets you:
- Search for any song or artist
- Play 30-second previews instantly
- Save your favorite tracks
- Create custom playlists
- Organize your music

All your data stays in your browser - no accounts, no signup required.

## Features

- **Search** - Find songs from iTunes library
- **Favorites** - Heart songs you like
- **Playlists** - Create and manage playlists
- **Queue** - See what's playing next
- **Controls** - Shuffle, repeat, volume control
- **Views** - Switch between grid and list layout

- Needs internet connection
